# Alke Wallet Appay

Proyecto académico desarrollado como parte del módulo **Fundamentos del Desarrollo Web y Bases de Datos**, cuyo objetivo es simular el funcionamiento básico de una **billetera digital (e-wallet)

---

## Funcionalidades principales

**Inicio de sesión** con validación de credenciales (simulada).
**Visualización y gestión de saldo**.
**Depósito de fondos**.
**Envío de dinero a contactos**.
**Historial de transacciones** (depósitos y transferencias).
**Diseño responsivo** adaptable a distintos dispositivos.

---

## Tecnologías utilizadas

* **HTML5** → Estructura semántica de la aplicación.
* **CSS** → Estilos personalizados y diseño visual.
* **JavaScript** →  Manejo de eventos.
* **Bootstrap 5** → Diseño responsive y componentes UI.
* **jQuery** → Manipulación del DOM, animaciones y validaciones.

---

## Flujo de navegación

1. **login.html** → Validación de acceso.
2. **menu.html** → Visualización del saldo y accesos rápidos.
3. **deposit.html** → Simulación de depósito.
4. **sendmoney.html** → Envío de dinero a contactos.
5. **transactions.html** → Consulta de movimientos recientes.

---

## Objetivos del proyecto

* Comprender la arquitectura básica de una aplicación web.
* Aplicar buenas prácticas de desarrollo frontend.
* Implementar interactividad con JavaScript y jQuery.
* Diseñar interfaces responsive orientadas a una fintech.
* Simular operaciones comunes de una billetera digital.

---

## Consideraciones importantes

* Este proyecto **no utiliza backend** ni persistencia real de datos.
* Toda la información es simulada en el navegador.
* El objetivo es educativo, no productivo.
  

---

## Posibles mejoras futuras

* Integración con backend.
* Persistencia de datos con base de datos.
* Autenticación real.
* Manejo de usuarios múltiples.
* Mejoras en seguridad y validaciones.
